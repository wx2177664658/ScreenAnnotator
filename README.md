# 屏幕标注白板（Windows）

Windows 桌面透明叠加标注工具。在 PPT、浏览器等演示内容上层实时画笔/图形标注，支持穿透操作下层窗口，并保存工程、导出 PNG、截屏带标注。

按 `docs/PRD-桌面屏幕标注白板.md` 及后续改版单实现。技术栈：WPF + .NET 8。

## 安装版（推荐，CR-009）

面向普通用户的主交付物：**Windows 安装程序**（Inno Setup，中文向导）。

| 项 | 说明 |
|----|------|
| 安装包 | `dist/ScreenAnnotator-Setup-1.0.8.exe` |
| 默认目录 | `%LocalAppData%\Programs\ScreenAnnotator\`（每用户，免管理员） |
| 运行时 | **自包含**，目标机无需预装 .NET |
| 开始菜单 | 安装后有「屏幕标注白板」快捷方式 |
| 桌面图标 | 向导中**可选**（默认不勾选） |
| 卸载 | 「应用和功能」或开始菜单「卸载 屏幕标注白板」 |
| 配置保留 | 卸载**不删除** `%APPDATA%\ScreenAnnotator\` 与用户自己的 `.board` |
| 升级 | 同安装包重复安装会覆盖，不产生两个入口 |
| 签名 | 当前未代码签名；首次可能被 SmartScreen 提示，选「仍要运行」即可 |

### 从源码打安装包

需已安装 [Inno Setup 6](https://jrsoftware.org/isinfo.php)，然后：

```bat
installer\build-setup.bat
```

或分步：

```bat
dotnet publish src\ScreenAnnotator\ScreenAnnotator.csproj -c Release -r win-x64 --self-contained true -o publish\ScreenAnnotator
"%LocalAppData%\Programs\Inno Setup 6\ISCC.exe" installer\ScreenAnnotator.iss
```

## 正确用法（怎么画）

1. 确认当前为**标注模式**（顶栏可见且选中「标注」；不是穿透）。
2. 在吸顶工具栏选工具后，在屏幕上**拖拽**绘制；文字工具点击空白落点输入。
3. **移动对象**：选工具栏「移动」，或**右键短按**打开移动模式后再拖；默认移动模式为关，绘制工具下不会悬停抢拖。右键打开移动模式时若当前是橡皮/框选删除，会自动切到「移动」工具。
4. **多选移动**：右键**长按拖拽**框选多个对象；开移动模式或「移动」工具后，拖任一选中项可整组平移；多选时可一起改颜色/粗细。
5. **调大小**：选中后拖角点（矩形/圆/笔迹/文字）或直线/箭头端点；圆默认保持正圆；文字可双击改文案；多选可整体等比缩放。
6. **删除**：选中后按 `Delete` / `Backspace` 删除（可撤销）；编辑文字内容时退格只改字。
7. **改样式**：选中（含多选）后，改颜色/粗细/字号作用于选中对象。
8. **橡皮**：对象橡皮单击删整段；笔刷橡皮拖过笔迹局部擦；框选删除拖框删相交对象。
9. 顶栏「收起」只留「展开」小按钮；穿透后小「展开」仍可点回标注。
10. 鼠标移入顶栏应为普通箭头且可点。


若按上述操作仍完全画不出笔迹，请反馈（见 `docs/BUG-001-无法绘制.md`）。

## 内容形态（屏幕标注 / 空白白板）

| | **屏幕标注（A）** | **空白白板（B）** |
|--|------------------|------------------|
| 背景 | 透明，透出桌面/PPT | 不透明白底 |
| 穿透 | 支持（笔迹仍可见） | **不点穿桌面**；「穿透」按钮改为「收起」 |
| 笔迹 | 独立一套 | 独立一套，切换互不影响 |
| 清屏 | 只清当前形态，且有二次确认 | 同左 |

默认启动为模式 A。保存 `.board`（version 2）同时写入两套对象；旧文件打开时仅恢复为 A，B 为空。

## 交互状态（标注 / 穿透·收起）

| 状态 | 笔迹 | 完整工具栏 | 顶部小入口 | 鼠标 |
|------|------|------------|------------|------|
| **标注** | 可见 | 可展开/收起 | 收起时显示 | 在本应用绘制 |
| **穿透（仅 A）** | **仍可见** | 隐藏 | **仍显示** | 点击穿透到下层 |
| **收起（B）** | 可见 | 隐藏 | 仍显示 | **不点穿**，仍挡在白板上 |

- 工具栏「标注 / 穿透(收起)」、`Ctrl+Shift+A`、托盘「显示/隐藏」三者状态同步
- 清屏（按钮或 `Ctrl+Shift+C`）：有笔迹时弹确认；确认后仍可撤销

## 多显示器

工具栏「屏幕」下拉可选当前系统显示器。选中后：

- 标注窗与吸顶工具栏跟随该屏
- **截屏带标注**打在**当前选中屏**（勿再假定主屏）
- 已有笔迹坐标不换算（左上对齐）；切到更小屏可能裁切

## 工具

画笔、**临时画笔**（松手约 0.3s 淡出，不保存）、荧光笔、直线、箭头、**形状**（矩形/圆角矩形/圆/三角/菱形/五边/六边/五角星/平行四边形/梯形）、文字、移动、对象/笔刷/框选橡皮、撤销/重做、清屏、取色、粗细（细/中/粗）、字号（小/中/大，与粗细独立）。移动模式/移动工具下选中对象可**旋转**。

## 默认快捷键

| 功能 | 默认快捷键 |
|------|------------|
| 显示/隐藏标注层 | `Ctrl+Shift+A` |
| 清屏（有笔迹时二次确认） | `Ctrl+Shift+C` |
| 撤销 | `Ctrl+Z` |
| 重做 | `Ctrl+Y` |
| 保存 | `Ctrl+S` |
| 开始/停止录屏 | `Ctrl+Shift+R` |
| 开/关移动模式 | **右键短按**（不可配置；设置页已移除该项） |
| 删除选中 | `Delete` / `Backspace` |
| 取消当前绘制/文字 | `Esc`（应用内） |

### 修改快捷键

1. 工具栏「设置」或托盘「快捷键设置」
2. 点击「录制」→ 按下新快捷键或**鼠标中键/侧键/滚轮**，**或直接在输入框手动填写**（如 `F6`、`MouseMiddle`、`滚轮上`、`Ctrl+鼠标中键`）
3. 支持单键、F1–F24、Ctrl/Shift/Alt/Win 组合，以及中键/侧键/滚轮（及带修饰键的鼠标组合）
4. **单独左键/右键不可绑定**（右键已用于移动/框选）；全局鼠标快捷键通过低级钩子实现，退出时卸载
5. 「保存并应用」后立即生效，写入 `%APPDATA%\ScreenAnnotator\settings.json`
6. 「恢复默认」可一键还原

若注册失败（与系统或其他软件冲突，**全局单键/鼠标较易冲突**），会提示更换。两项设成同一手势会在保存前拦截。Esc 仅用于取消录制，不可注册为业务键。

## 文件与导出

| 操作 | 说明 |
|------|------|
| 保存 / 打开 | `.board`（JSON）工程；v2 含 A/B 两套对象与选中屏；兼容旧单列表 |
| 导出 PNG | **当前形态**：A 透明底笔迹；B 白底+笔迹 |
| 截屏 | A：选中屏桌面 + A 笔迹；B：白板全屏图（不抓桌面） |
| 录屏 | 选中屏所见即所得（含标注）；可选麦/系统声（默认关）；输出 MP4 |

> 录屏开麦或系统声时，需本机有 `ffmpeg.exe`（可放到绿色包 `ffmpeg\` 目录，或系统 PATH）。纯画面录制不需要。

分辨率变化打开工程时：左上对齐，并提示尺寸不一致。

## 系统托盘

- 快捷键设置
- 退出

- 快捷键等设置：`%APPDATA%\ScreenAnnotator\settings.json`
