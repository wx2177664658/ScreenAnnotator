# 改版单：多显示器选屏（CR-005）

> **来源**：验收后增强  
> **优先级**：P0（本轮与 CR-007、CR-008 一起做）  
> **状态**：待开发；**策划仅单屏——多屏由开发双屏自测；策划单屏冒烟**  
> **关联**：PRD §5.1；本轮并行：CR-007 清屏确认、CR-008 独立白板  

---

## 1. 目标

用户可选择标注叠加在 **哪一台显示器** 上，而不仅是主屏。

---

## 2. 功能规格

### 2.1 入口

- 展开工具栏增加 **「屏幕」** 下拉或按钮：列出当前系统显示器，如 `显示器 1（主）1920×1080`、`显示器 2 …`。
- 托盘「切换屏幕」子菜单：加分，非必须。

### 2.2 行为

- 选中某屏后：
  - 标注窗（及模式 B 白板窗，见 CR-008）覆盖该屏；
  - 吸顶工具栏 / 小入口跟随到该屏顶部；
  - 已有笔迹：**坐标不换算、随画布到新屏**（左上对齐）；更小屏可裁切，README 说明。
- 不要求按比例映射分辨率。

### 2.3 单屏

- 仅一项时可选中、无副作用，供策划冒烟。

### 2.4 热键 / 截屏

- 热键作用于当前标注所在屏窗口。
- **截屏带标注** = **当前选中屏** + 当前模式笔迹（勿写死 Primary）。

### 2.5 与 CR-008

- 切换屏幕时，模式 A / 模式 B 都跟到目标屏；两套笔迹数据仍按 CR-008 隔离规则处理。

## 3. 验收

| 角色 | 标准 |
|------|------|
| 开发 | 双屏：切换后窗体、工具栏、绘制、截屏均在目标屏；切回正常 |
