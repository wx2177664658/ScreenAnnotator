# 改版单：临时画笔（CR-016）

> **状态**：已交付（开发完成，待策划验收）  
> **确认**：松手后 **0.2–0.4 秒淡出** 再消失（选项 A）  

---

## 1. 产品意图

快速讲解时指示「讲到哪里」：跟手可见，**松手后淡出消失**，不留永久笔迹，避免板面杂乱。

| | 普通画笔 | 临时画笔 |
|--|----------|----------|
| 拖动中 | 留下笔迹 | 留下临时笔迹 |
| 松手后 | 永久保留 | **0.2–0.4s 淡出后清除** |
| 保存/撤销 | 参与 | **不参与** |

---

## 2. 功能规格

### 2.1 入口
- 工具栏新增 **「临时画笔」**（建议紧挨「画笔」）。
- 颜色、粗细：使用当前工具栏颜色/粗细。

### 2.2 行为
1. 按下拖动：实时绘制临时笔画（可略高于画笔透明度以便醒目，非必须）。
2. 松手：该笔画在 **0.2–0.4 秒内淡出** 后完全移除。
3. 不写入对象列表；不进撤销栈；不进 `.board`；导出 PNG 不含已消失临时笔。
4. 若淡出未完成又开新一笔：可并行淡出或多笔各自计时；简单实现「新笔开始时清掉上一笔」也可，但推荐各笔独立淡出。
5. 录屏：拖动及淡出期间可见内容应能被录到。

### 2.3 其它
- 模式 A/B 均可用；穿透下不可绘。
- 切换工具时：未完成的临时笔立即取消或快速淡出。

### 2.4 非目标
- 一键把临时笔固化为永久笔（可后续）。
