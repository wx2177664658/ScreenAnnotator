# 改版单：帮助说明（CR-017）

> **状态**：已交付（开发完成，待策划验收）  
> **目标**：工具栏增加「帮助」，用简短中文说明主要操作，降低首次使用成本。  

---

## 1. 入口

- 展开工具栏增加 **「帮助」** 按钮（可靠近「设置」）。
- 托盘菜单增加「帮助」：加分，非必须。
- 点击后打开 **帮助窗口**（或滚动面板）：中文、可关闭；`Topmost` 以免被标注层挡住；不要全屏遮死授课画面（建议固定宽度约 420–520px 的对话框）。

---

## 2. 内容结构（须覆盖，文案可微调）

帮助内用小标题 + 短句即可，避免长文。建议分节：

### 2.1 两种模式
- **屏幕标注**：透明叠在桌面/PPT 上画。
- **空白白板**：不透明白底，纯板书。
- 顶栏可切换；两套笔迹互不影响。

### 2.2 标注与穿透
- **标注模式**：可画。
- **穿透 / 收起**：屏幕标注下可操作下层窗口，笔迹仍可见；空白白板下为收起工具栏，不能点穿桌面。
- 默认快捷键提示：`Ctrl+Shift+A`（以设置里实际默认为准，或写「见快捷键设置」）。

### 2.3 画笔与临时画笔
- **画笔**：松手后笔迹保留。
- **临时画笔**：讲解指位用，松手后约 0.3 秒淡出消失，不留板、不保存。

### 2.4 形状与线条
- **形状**：点「形状」展开（矩形、圆、多边形等），拖拽绘制。
- 直线、箭头在一级工具。

### 2.5 移动 / 缩放 / 旋转
- **右键短按**：开/关移动模式。
- 移动模式或「移动」工具下：拖对象移动；拖角缩放；拖旋转柄旋转。
- **右键长按拖拽**：框选多个，再一并移动。
- **Delete / Backspace**：删除选中（编辑文字时除外）。

### 2.6 橡皮
- 对象橡皮 / 笔刷橡皮 / 框选删除：各一句说明。

### 2.7 清屏 / 保存 / 导出 / 截屏 / 录屏
- 清屏有确认，可撤销。
- 保存工程、导出 PNG、截屏带标注、录屏（含标注；麦/系统声可选且默认关）各一句。

### 2.8 屏幕
- 「屏幕」下拉：选在哪块显示器上标注。

### 2.9 快捷键
- 打开「设置」可改快捷键；支持键盘与部分鼠标键（中键/侧键/滚轮等）；单独左右键一般不可绑业务键。
- 列出当前默认常用键（显示/隐藏、清屏、撤销、录制等）——**建议从配置/默认表动态生成**，避免文档与代码不一致；若静态写死须与 `DefaultHotkeys` 同步。

### 2.10 工具栏
- 吸顶；可收起，顶部小按钮展开。
- 颜色/粗细/字号等点击展开。
