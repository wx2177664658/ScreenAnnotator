# 改版单：快捷键支持鼠标按键（CR-011）

> **来源**：CR-010 之后优化  
> **优先级**：P0  
> **状态**：已交付
> **关联**：CR-010 键盘快捷键任意化（已交付）；CR-006/010 右键短按/长按手势  

---

## 1. 目标

快捷键设置除键盘外，还可绑定 **鼠标按键 / 滚轮**，例如：

| 手势名（存储/显示建议） | 含义 |
|-------------------------|------|
| `MouseLeft` | 左键 |
| `MouseRight` | 右键 |
| `MouseMiddle` | 中键 |
| `MouseXButton1` | 侧键 1（后退键） |
| `MouseXButton2` | 侧键 2（前进键） |
| `WheelUp` | 滚轮向上 |
| `WheelDown` | 滚轮向下 |
| `Ctrl+MouseMiddle` 等 | 修饰键 + 鼠标（须支持） |

录制：进入「录制」后，用户按下鼠标键或拨动滚轮即可录入（与键盘录制同一流程）。  
手动输入：允许键入上表名称（大小写不敏感，可用 `MouseRight` / `右键` 中文别名二选一，实现时统一一种并在 UI 提示）。

---

## 2. 技术约束（开发必读）

Windows `RegisterHotKey` **只支持键盘，不支持鼠标**。  
因此鼠标手势必须另通道，例如：

- 全局：`WH_MOUSE_LL` 低级鼠标钩子；或  
- 仅本应用前台时在窗口内处理（**不达标**——课堂常在 PPT 前台，全局热键类能力应对鼠标同样尽量全局）

策划要求：**鼠标快捷键与键盘快捷键一样，在其它应用前台时尽量仍可用**（钩子方案）。钩子需注意：

- 进程退出时卸载钩子  
- 与杀软/权限冲突时提示「鼠标快捷键注册失败」  
- 性能：钩子回调尽量轻量  

键盘部分仍可用现有 `RegisterHotKey`；配置里用手势字符串区分类型，分发到不同注册器。

---

## 3. 与现有右键手势的冲突规则（重要）

产品已占用：

- 右键短按 → 切换移动模式  
- 右键长按拖 → 框选移动  

若用户把某业务动作也绑成 `MouseRight`：

| 规则 | 说明 |
|------|------|
| R1 | 保存时 **警告**：`右键已用于移动模式/框选，绑定为全局快捷键可能导致冲突，是否仍保存？`（是/否） |
| R2 | 若仍保存：全局快捷键触发时，**优先业务快捷键还是优先内置手势**须固定——策划定案：**内置移动/框选手势优先**（短按/长按逻辑不变），绑在右键上的业务快捷键 **仅在「未触发内置手势」时** 不要求再做；更干脆：**禁止将「切换标注层 / 清屏」等绑到 MouseRight**，保存直接拒绝并提示改用其它键。 |

**策划定案（按此实现）：**

- **允许**绑定：中键、侧键、滚轮、以及 `Ctrl+鼠标键` 等。  
- **默认禁止**单独绑定 `MouseLeft`、`MouseRight` 为全局业务快捷键（左键会破坏点击，右键与内置手势冲突）。  
- 若用户强行要绑左/右键：仅允许带修饰键的组合，如 `Ctrl+MouseRight`、`Alt+MouseLeft`（保存时校验）。  
- 设置页用灰色说明写清上述规则。

---

## 4. 滚轮

- `WheelUp` / `WheelDown`：每次刻度触发一次对应动作（注意防抖，避免一次拨动触发多次——可按系统一次 notch 一次）。  
- `Ctrl+WheelUp` 等组合须支持。  
- 不要求支持「按住中键拖动」类手势（本单不做）。

---

## 5. UI

- 录制提示文案改为：`请按下键盘按键或鼠标键 / 拨动滚轮…（Esc 取消）`  
- 列表里鼠标手势显示为中文更佳：`鼠标中键`、`滚轮上`、`侧键1` 等（存储可用英文 token）。

---

## 6. 验收

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 录制中键绑到「清屏」或「显示/隐藏」 | 可保存；在其它窗口前台按下中键能触发（钩子生效） |
| 2 | 录制滚轮上/下 | 可绑定并触发 |
| 3 | 录制侧键 | 笔记本/鼠标有侧键时可测；无侧键可不测硬件 |
| 4 | 尝试单独绑左键/右键 | 被拒绝或要求加修饰键；有说明 |
| 5 | `Ctrl+中键` | 可录可触发 |
| 6 | 右键短按/长按框选 | 内置行为不回退 |
| 7 | 键盘单键/组合键（CR-010） | 不回退 |
| 8 | 卸载钩子 | 退出应用后鼠标恢复正常，无残留 |

## 7. 发给开发的话术

---

请做 CR-011，见 `docs/CR-011-快捷键支持鼠标按键.md`。

在 CR-010 键盘任意快捷键基础上：设置里可录制/绑定 **鼠标左/右/中/侧键、滚轮上下**，以及 `Ctrl+鼠标键` 等组合。注意 `RegisterHotKey` 不支持鼠标，需全局鼠标钩子（或等价方案）。

冲突：单独左键/右键默认禁止作业务快捷键（右键已用于移动/框选）；允许中键、侧键、滚轮及带修饰键的鼠标组合。右键短按/长按内置手势不回退。

按 §6 自测后发包。
