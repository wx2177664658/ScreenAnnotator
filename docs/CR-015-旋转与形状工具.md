# 改版单：移动模式旋转 + 形状工具扩展（CR-015）

## 1. 移动模式下旋转

### 1.1 目标
**移动模式开** 或工具栏为「移动」，且已选中对象时：在移动、缩放之外增加 **旋转**。

### 1.2 交互
- 选中后显示 **旋转手柄**（建议在包围盒顶边中点上方）。
- 拖拽：绕包围盒中心旋转；实时预览；松手 = 撤销一步。
- **Shift**（加分）：15° 吸附。
- 手柄光标为旋转样式；与缩放角点、直线端点命中分区清晰。

### 1.3 适用范围（单选必做）
矩形、圆角矩形、圆/椭圆、全部新多边形、直线、箭头、文字、画笔/荧光笔路径。  
多选整组旋转：加分。

### 1.4 存储与导出
- 保存旋转后外观（`rotation` 字段或烘焙坐标均可）。
- 导出 PNG / 截屏 / 录屏须为旋转后效果。

## 2. 形状工具栏

### 2.1 一级入口
- 取消一级平铺的「矩形」「圆形」。
- 新增一级 **「形状」**：点击展开面板/下拉。
- 按钮显示当前子形状；记住上次选择。

### 2.2 面板内形状清单（已定）

| 形状 | 说明 |
|------|------|
| 矩形 | 原有 |
| 圆角矩形 | 新 |
| 圆形 | 原有 |
| 三角形 | 新 |
| 菱形 | 新 |
| 五边形 | 新 |
| 六边形 | 新 |
| 五角星 | 新 |
| 平行四边形 | 新（第 8 项） |
| 梯形 | 新（第 8 项） |

直线、箭头仍留一级工具，不进形状面板。

### 2.3 绘制与编辑
- 对角拖拽外接矩形生成形状（与现矩形一致）。
- 选中后：移动、缩放（CR-012）、旋转（本单）、改色/粗细。
