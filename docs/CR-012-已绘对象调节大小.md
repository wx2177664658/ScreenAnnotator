# 改版单：已绘对象可调大小/长度等属性（CR-012）

> **来源**：定版前增强  
> **优先级**：P0  
> **状态**：已交付
> **关联**：CR-004/006 选中与移动；CR-010 多选；样式改色/粗细已有  

---

## 1. 目标

对**已经画好**的对象，除移动、改色/粗细外，还能调整其 **几何尺寸 / 长度 / 字号（可视大小）** 等属性，无需删了重画。

适用类型：矩形、圆形、箭头、直线（横线）、文字、画笔/荧光笔路径。

---

## 2. 进入编辑的前提

与现网选中规则一致：

- 移动模式开，或工具栏为「移动」；或已通过框选多选中的对象。
- **单击选中**后出现选中框 + **调节手柄（handles）**。
- 多选时：本轮 **只支持整体等比缩放**（包络框四角拖）；若工期紧，多选可仅允许移动+改色，**单选必须支持**下列调节（多选缩放为加分）。

---

## 3. 各类型可调属性与交互

### 3.1 矩形 / 圆形（椭圆）

| 属性 | 交互 |
|------|------|
| 宽高 / 外接矩形大小 | 选中后四角（及可选四边中点）手柄拖拽变形 |
| 位置 | 已有：拖动对象移动 |
| 粗细 / 颜色 | 已有：工具栏 |

- 圆形：拖角时建议 **保持正圆**（Shift 或默认保持；策划定：**默认保持圆**，按外接正方形缩放）；若实现困难可按椭圆自由拉。
- 手柄拖拽松手 = 撤销一步。

### 3.2 直线 / 箭头

| 属性 | 交互 |
|------|------|
| 端点位置（从而改变长度与方向） | 选中后 **两端点** 各一手柄，拖拽改起点/终点 |
| 线宽 / 颜色 | 已有工具栏 |
| 整体平移 | 拖线身（非端点）= 移动整段（现有移动） |

- 箭头头部随终点方向自动更新。

### 3.3 文字

| 属性 | 交互 |
|------|------|
| 字号（视觉大小） | ① 选中后改工具栏字号（已有）② **加**：选中框角点拖拽 → 连续缩放字号（有最小/最大限制，如 8–200） |
| 内容 | 本轮 **双击** 文字进入编辑（改文案）；Esc 取消，Enter/失焦确认。若双击与移动冲突：双击才进编辑，单击仍为选中。 |
| 颜色 | 已有 |

### 3.4 画笔 / 荧光笔

| 属性 | 交互 |
|------|------|
| 粗细 | 已有：选中后改工具栏粗细 |
| 整体大小（缩放路径） | 选中后包围盒 **四角拖拽等比缩放**（所有点相对中心或相对角点缩放） |
| 颜色 | 已有 |

- 不做「只拉长某一段笔迹」的局部变形（本轮非目标）。

---

## 4. 通用交互细则

1. 手柄命中优先于「开始新绘制」；光标在角点为斜向缩放光标，在线端点为可拖光标。  
2. 调节过程中实时预览，松手提交历史。  
3. 保存 `.board`：仍存矢量几何，打开后尺寸正确。  
4. 模式 A/B 两套对象均支持。  
5. README 补充：选中后拖角/端点可改大小；文字可双击改内容。

---

## 5. 非目标

- 旋转手柄（可后续）  
- 自由变形/透视  
- 矢量锚点级编辑笔迹  
- 圆角矩形等新类型  

---

## 6. 验收标准

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 选中矩形，拖角 | 宽高变化；可撤销 |
| 2 | 选中圆，拖角 | 尺寸变化（优先保持圆） |
| 3 | 选中直线/箭头，拖端点 | 长度/方向变；箭头朝向正确 |
| 4 | 选中文字，改字号或拖角 | 文字变大变小；双击可改文案 |
| 5 | 选中笔迹，拖包围盒角 | 整段等比缩放；改粗细仍可用 |
| 6 | 保存再打开 | 尺寸保持 |
| 7 | 回归 | 移动、框选、绘制、穿透不回退 |

---

## 7. 发给开发的话术

---

请做 CR-012，见 `docs/CR-012-已绘对象调节大小.md`。

选中已画好的矩形/圆/箭头/直线/文字/画笔后，除改色粗细外，须能调大小/长度等：角点缩放矩形圆与笔迹；直线箭头拖两端点；文字可改字号且建议支持双击改文案。单选必做；多选整体缩放加分。按 §6 自测发包。
