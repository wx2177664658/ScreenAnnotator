## 2. 悬停移动 + 点选改样式（无「选择」工具）

### 2.1 适用范围

以下对象均支持悬停移动（不限于矩形/线/文字，行为统一，降低学习成本）：

- 直线、箭头、矩形、圆形、文字  
- 画笔 / 荧光笔路径（命中笔画或包围盒边缘均可，实现取稳妥命中即可）

### 2.2 光标与命中

- 当前为 **绘制类工具**（画笔/荧光/线/箭头/矩形/圆/文字）且指针命中某对象的可移动区域时：
  - 光标切换为 **移动光标**（如 `SizeAll`）
  - **不要**显示画笔光标
- 可移动区域建议：
  - 矩形/圆：描边附近 + 包围盒边缘（可含少量内边距），避免整块填充内部都抢移动导致难在框内再画
  - 直线/箭头：线段附近容差带
  - 文字：文字包围盒
  - 路径：笔画容差或包围盒边
- 命中多个时：取 **最顶层（后绘制）** 一个
- 指针在空白处：恢复当前绘制工具光标，可正常开画

### 2.3 拖拽移动

- 在命中对象上 **按下并拖动超过轻微阈值**（如 3–4px）→ 进入移动，平移该对象全部几何
- 松手 = 历史 **一步**（可撤销）
- **不需要**先选中再拖；悬停直接拖即可
- 移动过程中可显示对象选中框，松手后 **保持选中**（便于接着改色）

### 2.4 单击选中（为改色/粗细服务）

因无选择工具，用单击区分：

| 操作 | 结果 |
|------|------|
| 在对象上单击（移动 &lt; 阈值） | **选中**该对象（显示选中框）；取消其他选中 |
| 在空白处单击 | 取消选中；若当前是绘制工具，不因此误画（按下移动才开画，或点击空白仅取消选中——绘制仍以拖拽/按下移动为准，与现逻辑对齐） |
| 拖拽移动 | 移动并选中该对象 |

选中态：浅色包围框即可；本轮不做缩放手柄。

### 2.5 选中后改色 / 粗细 / 字号

- **有选中**时：改颜色 / 粗细 / 字号 → 写回选中对象（粗细对文字可忽略；字号仅文字生效）
- **无选中**时：只影响后续新绘制（现有行为）
- 每次样式变更进撤销栈

### 2.6 与绘制的优先级（关键）

在 **画笔/图形/文字** 工具下：

1. 若按下时命中可移动对象 → **走移动/点选**，不新建笔画  
2. 若按下时在空白 → **走原有绘制**  

文字工具：空白处单击仍弹出输入；命中已有文字则选中/可拖移。

### 2.7 与橡皮类工具

- 当前为 **对象橡皮 / 笔刷橡皮 / 框选删除** 时：**不启用**悬停移动（避免与擦除冲突）
- 使用橡皮类时取消选中（或保留选中但不挡擦除，策划定：**取消选中**）

### 2.8 明确不做

- 工具栏「选择」工具  
- Ctrl 多选、选中后缩放/旋转

## 3. 橡皮增强

### 3.1 工具

| 工具名 | 行为 |
|--------|------|
| **对象橡皮** | 单击命中 → 删除整对象（保留现逻辑，可改名） |
| **笔刷橡皮** | 拖动局部擦路径；与线/箭头/矩形/圆/文字轨迹相交 → **整对象删除** |
| **框选删除** | 拖框，删除与框相交的对象；可撤销 |

工具栏过挤时允许合并为「橡皮」下的子模式，但默认要能发现「笔刷」「框选」。推荐三按钮。

### 3.2 笔刷橡皮（Q1-A）

- 对 pen/highlighter：删半径内点，必要时拆成多段；过短段丢弃  
- 半径跟随粗细三档或固定中档，README 说明  
- 按下到松开 = 历史一步  

### 3.3 框选删除

- 半透明框预览；松手删除相交对象；空框不删
