# 改版单：录屏（CR-014）

> **状态**：已交付（开发完成，待策划验收）  
> **确认**：Q1=A，Q2=B（麦+系统声均可选，**默认均关**），Q3=A，Q4=A

## 1. 目标

录制 **当前选中显示器** 上的教学过程，画面为 **所见即所得（含标注）**；音轨可选麦克风与系统声；输出 MP4。

---

## 2. 功能规格

### 2.1 画面
- 录制 CR-005 **当前选中屏** 的完整画面。
- **必须包含本应用标注层**（模式 A：桌面+标注；模式 B：白板+笔迹）。
- 开始录制后：**自动收起完整吸顶工具栏**，保留带 **红点 + 计时** 的小入口（可点停止）；避免挡内容。
- 回放验收标准：**能看见标注笔迹**。若系统捕获录不到透明顶层，开发须合成或改捕获方案，不得静默变成「无标注」。

### 2.2 音频（可选，默认关）
开始录制前（或录制设置小面板）提供两个开关：

| 开关 | 默认 | 说明 |
|------|------|------|
| **麦克风** | **关** | 讲解人声 |
| **系统声音** | **关** | 系统/软件播放声（PPT 内视频等） |

- 可只开麦、只开系统声、两个都开、两个都关（纯画面）。
- 两路都开时混音进同一 MP4 音轨（或双轨，实现任选，回放能听到即可）。
- 无设备/无权限：提示，不阻断纯画面录制。
- 首次开麦可能触发系统麦克风权限请求，需能处理。

### 2.3 控制
| 操作 | 行为 |
|------|------|
| 工具栏「录制」 | 打开简短设置（路径 + 两音源开关）或先选路径再开始；开始捕获 |
| 「停止」/ 红点入口 | 结束并完成写文件；提示成功与路径 |
| 快捷键 | 新增可配置项：开始/停止录制（建议默认 `Ctrl+Shift+R` 切换）；进快捷键设置 |
| 退出应用时若正在录 | 询问是否停止并保存；勿直接丢文件 |

录制中状态文案示例：`● 录制中 00:12:34`。

### 2.4 输出
- 格式：**MP4**（建议 H.264；有音则 AAC）。
- 每次开始前 **另存为选路径**；默认文件名如 `录屏_yyyyMMdd_HHmmss.mp4`。
- 帧率：默认 24 或 30；分辨率跟选中屏。

### 2.5 非目标（本单不做）
- 摄像头画中画、直播、云上传  
- 自选区域录制（非全屏当前显示器）  
- 暂停/继续（本单不做，仅开始/停止）  

---

## 3. 技术备注
- 可选：Windows Graphics Capture、Media Foundation、FFmpeg 等。
- 系统声采集（如 WASAPI loopback）与麦权限分开处理。
- 长时间录制注意磁盘与性能；退出卸钩/热键逻辑勿回退 CR-013。

---

## 4. 验收标准

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 画标注 → 录一段 → 停止 | 得到 MP4，回放有标注 |
| 2 | 麦、系统声均关 | 纯画面可播 |
| 3 | 只开麦 | 有人声 |
| 4 | 只开系统声 | 能录到系统播放声（环境允许时） |
| 5 | 两路都开 | 回放两路都能听到（或可辨） |
| 6 | 默认两开关均为关 | UI 默认状态正确 |
| 7 | 录制中顶栏收起，红点可停；快捷键可停 | 符合规格 |
| 8 | 选中副屏录制（有双屏时） | 录的是选中屏 |
| 9 | 回归 | 标注/退出卡顿修复不回退 |

## 5. 发给开发的话术

---

请做 CR-014 录屏，见 `docs/CR-014-录屏.md`（已拍板）。

要求摘要：
1. 录 **当前选中屏**，画面 **含标注**（所见即所得）；回放必须能看见笔迹。  
2. **麦克风**、**系统声音** 两个可选开关，**默认都关闭**；可只开其一或都开。  
3. 工具栏开始/停止；录制中收起顶栏，留红点计时可停；快捷键可配置（建议默认 Ctrl+Shift+R）。  
4. 输出 **MP4**，每次自选保存路径。  
5. 不做暂停、摄像头、区域录制。

按文档 §4 自测后更新绿色包并把路径发策划验收。
