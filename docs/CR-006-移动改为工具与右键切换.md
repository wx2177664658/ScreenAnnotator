# 改版单：移动改为工具/快捷键/右键切换（CR-006）

> **来源**：CR-004 验收通过后的体验优化  
> **优先级**：P0  
> **状态**：待开发实现  
> **关联**：废止 CR-004 §2 中「绘制工具下悬停即可移动」的行为；改色/橡皮等其余 CR-004 能力保留  

---

## 1. 问题

当前：绘制工具下，鼠标悬停对象框/边即进入可拖状态。  
副作用：想在对象附近或重叠区域**精确新标注**时，容易被移动抢走，画不准。

---

## 2. 目标行为（已拍板）

| 方式 | 行为 |
|------|------|
| **工具栏「移动」工具** | 新增独立工具；选中后才能靠左键拖对象移动 / 单击选中 |
| **快捷键** | 可切换到移动工具（或切换「移动模式」开/关）；进快捷键设置，可改，持久化 |
| **鼠标右键** | **默认**单击右键 **切换**「移动模式」开 ↔ 关（不依赖是否选中移动工具图标的另一种入口，见下） |

关闭移动能力后：当前绘制工具在对象上/旁均按**绘制**处理，**禁止**悬停变移动光标、禁止左键拖走对象。

---

## 3. 详细规格

### 3.1 废除

- 绘制类工具（画笔/荧光/线/箭头/矩形/圆/文字）下：**不再**因悬停对象而出现移动光标，**不再**左键拖对象。

### 3.2 工具栏「移动」工具

- 名称：**移动**（或「移动/选择」）。
- 选中该工具时：
  - 悬停可命中对象 → 移动光标；
  - 左键拖 → 移动；单击 → 选中（改色/粗细/字号逻辑与 CR-004 相同）；
  - 空白处单击 → 取消选中；空白处不绘制。
- 切回画笔等工具 → 退出「仅工具栏选中的移动」，但若 **右键移动模式仍为开**，见 3.4。

### 3.3 快捷键（P0）

- 新增动作 ID，建议：`toggle_move_mode` 或 `tool_move`。
- **默认快捷键（建议）**：`Ctrl + Shift + M`（若冲突可在设置里改）。
- 行为（二选一，策划定案如下）：
  - **切换移动模式开/关**（与右键同一套状态），打开时等价于可移动，关闭后恢复绘制优先。
- 必须出现在现有「快捷键设置」中：可录制、冲突提示、恢复默认、持久化、即时生效（与现网热键体系一致）。

### 3.4 右键切换「移动模式」（P0，默认手势）

- **右键单击**（不拖拽）：切换全局标志 `MoveModeEnabled`。
  - **开**：即使工具栏当前是画笔等，也允许「悬停移动光标 + 左键拖移/点选」（与选中「移动」工具时相同的命中移动逻辑）。
  - **关**：仅当工具栏明确选中「移动」工具时才能移动；否则一律绘制优先，无悬停抢夺。
- 右键**不要**再用于「取消当前绘制」若与切换冲突；取消绘制改由 `Esc` 承担（现网已有）。
- 建议：移动模式 **开** 时，工具栏「移动」按钮呈选中/高亮，或单独指示灯（如状态栏「移动：开」），避免用户不知道当前能不能拖。
- 右键切换后给一次轻反馈即可（状态文案变化）；不必弹窗。

### 3.5 状态叠加（避免歧义）

| 工具栏当前工具 | MoveModeEnabled（右键/快捷键） | 左键在对象上 |
|----------------|----------------------------------|--------------|
| 移动 | 任意 | 移动 / 点选 |
| 画笔等绘制 | **开** | 移动 / 点选 |
| 画笔等绘制 | **关** | **绘制**（本单核心） |
| 橡皮 / 笔刷橡皮 / 框选删 | 任意 | 仍走橡皮逻辑，不移动 |

说明：右键「开」= 临时允许在绘制工具下移动（用户主动打开）；默认启动时 **MoveModeEnabled = 关**，避免再出现「一悬停就拖走」。

### 3.6 改色 / 粗细

- 仍依赖「选中对象」：在可移动状态下单击选中后改样式（逻辑不变）。

### 3.7 README

- 写明：移动用「移动」工具，或右键开/关移动模式，或快捷键；默认右键切换；关闭后可精确标注。

## 5. 验收标准

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 启动，画笔，MoveMode 默认关，悬停已有矩形 | **不变**移动光标；可在旁/上准确新画 |
| 2 | 点工具栏「移动」，拖对象 | 可移动；可单击选中改色 |
| 3 | 切回画笔，MoveMode 仍关 | 不能拖对象，只能画 |
| 4 | 画笔下右键一下 | 进入移动开；悬停可拖；再右键关闭 |
| 5 | 默认快捷键切换移动模式 | 与右键同一状态，设置里可改 |
| 6 | 橡皮工具下 | 不移动，只擦/框删 |
| 7 | 回归 | 绘制、穿透、工具栏点击、CR-004 橡皮与改色不回退 |
