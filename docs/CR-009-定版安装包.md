# 改版单：定版安装包（CR-009）

> **来源**：功能验收收口后定版  
> **优先级**：P0（发布交付）  
> **状态**：已交付（Inno Setup；产物 `dist/ScreenAnnotator-Setup-1.0.0.exe`）  
> **关联**：原搁置「安装包 / MSI」；绿色包 `publish/ScreenAnnotator` 仍可保留作便携版  

---

## 1. 目标

提供面向普通用户的 **Windows 安装程序**：双击安装、开始菜单可开、可卸载；目标机 **无需预装 .NET**（沿用现网自包含发布）。

绿色解压包可继续提供，但 **定版主交付物 = 安装包**。

---

## 2. 产品信息（安装器用）

| 项 | 值 |
|----|-----|
| 显示名称 | 屏幕标注白板 |
| 建议安装名 / 快捷方式名 | 屏幕标注白板 |
| 可执行文件 | `ScreenAnnotator.exe` |
| 默认安装目录 | `%LocalAppData%\Programs\ScreenAnnotator\` 或 `C:\Program Files\ScreenAnnotator\`（二选一；**推荐每用户 LocalAppData**，免管理员） |
| 发行版本号 | 由开发定 `1.0.0`（或按现有程序集版本）；安装包与 About/文件属性一致 |
| 发布说明 | README 简述功能与快捷键 |

---

## 3. 安装包能力（必做）

| # | 能力 |
|---|------|
| 1 | 一键/向导安装（中文界面优先） |
| 2 | 写入开始菜单快捷方式 |
| 3 | **可选**桌面快捷方式（默认勾选或默认不勾，实现时选一种并在向导写清） |
| 4 | 安装自包含运行所需全部文件（与当前 `dotnet publish -r win-x64 --self-contained` 一致） |
| 5 | 控制面板 /「应用和功能」中可 **卸载**，卸净程序文件与快捷方式 |
| 6 | 卸载 **不要删除** 用户配置（`%APPDATA%\ScreenAnnotator\`）与用户自己的 `.board`；可在卸载说明中注明 |
| 7 | 重复安装 / 升级：覆盖同目录，不出现两个并列混乱入口（同一 UpgradeCode / 应用 ID） |
| 8 | 产出物放仓库约定目录，如 `dist/ScreenAnnotator-Setup-1.0.0.exe`（或 `.msi`） |

### 技术选型（开发自定，须文档化）

任选其一即可，以稳定、免额外运行时为准：

- **Inno Setup**（常见、中文好做）  
- **WiX / MSI**  
- **MSIX**（若环境熟）

策划不强制 MSI；**有安装向导 + 可卸载** 即达标。

## 5. 验收标准

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 在干净或普通 Win10/11 机双击安装包 | 可完成安装，中文可懂 |
| 2 | 开始菜单打开 | 应用正常启动，主功能可用 |
| 3 | 不预装 .NET | 仍能运行（自包含） |
| 4 | 卸载 | 程序与快捷方式移除；设置目录可保留 |
| 5 | 再装一遍 | 正常覆盖/升级，能打开 |
| 6 | 仓库/发布目录 | 有明确安装包文件 + 简短《安装说明》（可写在 README「安装版」一节） |

策划本机按 1–5 冒烟即可。
