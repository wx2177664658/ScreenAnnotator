# 改版单：快捷键任意化 + 右键长按框选移动（CR-010）

> **来源**：定版前体验优化  
> **优先级**：P0  
> **状态**：已交付
> **关联**：CR-006 右键短按切换移动模式；CR-009 安装包可并行或其后  

---

## 0. 右键交互（策划默认，可改）

| 手势 | 行为 |
|------|------|
| **右键短按**（按下很快抬起，无明显拖拽） | **保留**：切换移动模式开/关（CR-006） |
| **右键长按后拖拽** | **新增**：框选；选中多个对象，用于一并移动 |

若产品要改成「取消短按切换、右键只做框选」，策划另行通知；**未通知前按上表实现**。

**长按判定建议**：按下持续 ≥ **300ms** 且移动超过阈值，或按下后拖拽距离 ≥ **5px** 即视为框选（两种满足其一即可，避免和短按切换打架）。

## 1. 快捷键设置不限制格式

### 1.1 现状问题

录制时若无 Ctrl/Shift/Alt，会提示「请使用包含 Ctrl / Shift / Alt 的组合键」，用户无法设单键（如 `F6`、`M`）等。

### 1.2 目标

- 用户可设置 **任意合理快捷键**：单键、F1–F24、带修饰键组合、Win 组合等。
- 录制时 **取消**「必须含修饰键」的拦截。
- 设置页手势框支持 **手动输入**（如 `Ctrl+Alt+Q`、`F8`、`Shift+Space`），保存时解析；格式尽量宽松。
- 应用内冲突（两项同一手势）仍禁止保存并提示。
- 系统 `RegisterHotKey` **注册失败**时：保存时可先写入配置，注册失败提示「注册失败请更换」（与现网一致）；不要因「没有修饰键」在录制阶段直接拒绝。
- Esc：仍用于取消本次录制，不把 Esc 录成业务快捷键（除非产品明确要；本单 **Esc 不作为可注册业务键**）。
- 纯修饰键（只有 Ctrl）仍无效，提示「请再按一个主键」。

### 1.3 解析增强

在现有 `TryParseGesture` / `FormatGesture` 基础上，尽量支持：

- 无修饰：`F6`、`A`～`Z`、`0`～`9`、以及 `Key` 枚举能解析的常见键（如 `Space`、`Oem*` 等）
- 有修饰：任意 `Ctrl/Shift/Alt/Win` 组合 + 主键

README 注明：全局单键可能与系统或其他软件冲突，失败请换键。

### 1.4 验收

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 录制 `F6` 或单字母 | 可保存，不再被「必须组合键」拦住 |
| 2 | 手动输入合法手势并保存 | 能解析并尝试注册 |
| 3 | 两项设成同一键 | 保存前冲突提示 |
| 4 | 注册失败 | 有失败提示，可改键 |
| 5 | 默认组合键 | 行为不回退 |

## 2. 框选移动（默认右键长按框选）

### 2.1 目标

支持 **多选** 后一起移动；默认用手势：**右键长按拖出选框**。

### 2.2 框选行为

1. 右键长按（见 §0）并拖拽 → 出现半透明选框（可与「框选删除」外观区分，如蓝色系 vs 删除用红色系）。
2. 松手：选中所有与选框 **相交** 的当前模式对象（多选）。
3. 选中态：每个对象显示选中框，或统一包络框 + 计数提示（实现任选，需能看出多选）。
4. **移动**：
   - 在 **移动模式开** 或工具栏为「移动」时：左键在任一选中对象上拖拽 → **整组平移**（相对位移相同）；
   - 移动模式关且非移动工具：框选仍可完成多选，但左键优先绘制（与 CR-006 一致）；用户需开移动模式或切移动工具后再拖。
5. 空白处左键单击：取消多选（绘制规则不变）。
6. 改色/粗细：有多选时，样式变更作用于 **所有选中对象**（文字字号仅改文字对象）。

### 2.3 与「框选删除」区分

| | 框选移动（本单） | 框选删除（已有） |
|--|------------------|------------------|
| 入口 | 右键长按拖（默认）；可选：移动工具下左键拖空白也框选（加分） | 工具栏「框选删除」 |
| 结果 | 多选，可再移动/改样式 | 直接删除 |

### 2.4 加分（非必须）

- 移动工具下，左键空白拖拽 = 框选（不删）。
- 状态栏提示「已选 N 个」。

### 2.5 验收

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 右键短按 | 仍切换移动模式（若未改 §0） |
| 2 | 右键长按拖框盖住多个对象 | 多选成功 |
| 3 | 开移动模式后左键拖其中一个 | 整组移动；可撤销 |
| 4 | 多选后改颜色 | 选中项都变色 |
| 5 | 短按与长按不严重误触 | 短按不误进框选；框选不误切模式 |
| 6 | 回归 | 框选删除、单对象移动、绘制不回退 |

## 4. 发给开发的话术

---

请做 CR-010，见 `docs/CR-010-快捷键任意化与框选移动.md`。

1. **快捷键**：去掉「必须 Ctrl/Shift/Alt」限制，允许单键/F 键等任意合理手势；支持手动输入；冲突与注册失败仍提示。  
2. **框选移动**：默认 **右键长按拖拽** 框选多对象；开移动模式或「移动」工具下可整组拖移；多选可一起改色/粗细。右键短按仍切换移动模式（长按/短按要区分好）。
